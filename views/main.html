Examples

Grid lets you jump straight into game logic.

Making a HUD element 

module( "gui.hudhealth", package.class, package.inherits "gui.panel" )

function _M:draw()
	love.graphics.print( localplayer:getHealth() )
	gui.panel.draw( self )
end

Making an entity 

module( "npc_monster", package.class, package.inherits "npc" )

function _M:npc_monster()
	npc.npc( self )
	self:setSprite( "images/monster.png" )
end

entities.linkToClassname( _M, "npc_monster" )

Playing a sound from an entity 

function _M:onTakeDamage()
	npc.onTakeDamage( self )
	self:emitSound( "sounds/cry.wav" )
end
